# 🚀 DSA Smart Extension

An AI-powered Chrome extension that helps users solve DSA problems by providing progressive hints instead of full solutions.

---

## ✨ Features

- 🔍 Extracts problem statements using DOM scraping  
- 🤖 Generates step-by-step hints using AI (NVIDIA NIM API)  
- 🧠 Helps users think instead of giving direct answers  
- 🌐 Supports multiple platforms:
  - LeetCode
  - HackerRank  

---

## 🛠️ Tech Stack

- JavaScript  
- Chrome Extensions API  
- DOM Manipulation  
- NVIDIA NIM API  

---
